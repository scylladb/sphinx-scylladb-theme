Nodetool cfstats
================
**cfstats** ``[<keyspace.cfname>...]`` - This command provides in-depth diagnostics regard table. The cfstats (column family statistics) option shows a high level statistics summary of the table.

For example:

.. code-block:: shell

   nodetool cfstats
