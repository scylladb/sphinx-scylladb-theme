==============
Search support
==============

The toolchain uses the service `Expertrec <https://expertrec.com/>`_ to offer a unified search experience across all sites.

Installation
------------

.. note::
    The process of listing a new site must be done by an Expertrec administrator.
    Contact us in Slack (``#scylla-docs`` channel) to set this configuration for you.

For more information, see `Crawl new URLs <https://blog.expertrec.com/knowledge-base/urls/>`_.
