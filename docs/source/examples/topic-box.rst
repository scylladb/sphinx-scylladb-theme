Topic Box

Default Topic
-------------

With Icon
=========

Using:

.. code-block:: rst

    .. topic-box::
        :title: Lorem Ipsum
        :icon: fa fa-github
        :link: #
        :anchor: Lorem ipsum

        Lorem ipsum dolor sit amet, consectetur adipiscing elit.

Results in:

.. topic-box::
    :title: Lorem Ipsum
    :icon: fa fa-github
    :link: #
    :anchor: Lorem ipsum

    Lorem ipsum dolor sit amet, consectetur adipiscing elit.
