Tables
======

We use the List Table directive for tables.
If possible give your table a name as this helps with SEO and screen readers for Accessibility.
In the width declaration, determine the number of columns and the width for each.

Regular table
-------------

Using:

.. code-block:: rst

    .. list-table:: Caption
        :widths: 33 33 33
        :header-rows: 1

        * - header name
            - header name
            - header name
        * - body text
            - body text
            - body text

Results in:

.. list-table:: Caption
    :widths: 33 33 33
    :header-rows: 1

    * - header name
      - header name
      - header name
    * - body text
      - body text
      - body text
