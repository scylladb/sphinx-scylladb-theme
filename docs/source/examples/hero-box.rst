========
Hero Box
========

Default
-------

Using:

.. code-block:: rst

    .. hero-box::
        :title: Sphinx ScyllaDB Theme

        The documentation toolchain for ScyllaDB projects.

Results in:

.. hero-box::
    :title: Sphinx ScyllaDB Theme

    The documentation toolchain for ScyllaDB projects.

----------


Using:

.. code-block:: rst

    .. hero-box::
        :title: Sphinx ScyllaDB Theme
        :image: /_static/img/mascots/scylla-enterprise.svg

        The documentation toolchain for ScyllaDB projects.

Results in:
