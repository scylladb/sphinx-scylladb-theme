Reference documentation

Scylla is compatible with Apache Cassandra, version 2.1.8, and can be used with current Cassandra drivers.

Scylla is a high-performance program and benefits greatly from configuration in the underlying OS platform. See the System Configuration Guide for information on how to configure your OS for Scylla.
